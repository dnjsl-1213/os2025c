Go 명령어
go mod init 프로젝트폴더명 : 현재 디렉토리를 Go모듈로 초기화
go run 파일명.go : 컴파일 + 실행까지 한번에 함
go build 파일명.go : 실행파일로 빌드

go get 깃허브주소 : 깃허브에서 패키지를 가져오는 명령어


리눅스 명령어
cp 원본파일 복사본파일 : 파일 복사


깃 명령어
git reset : 커밋 이력이나 스테이징 상태를 되돌림
git merge 브랜치명 : 다른 브랜치의 내용을 현재 브랜치에 병합


기타
.gitignore 파일 생성 (*.exe) : Git에서 추적하지 않을 파일/폴더를 지정(파일에 .exe를 넣으면 .exe파일은 git에 올라가지 않음) / 무시
package, import, func, Println, Printf, {}, ;
